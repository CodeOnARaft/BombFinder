# Bomb Finder
## Overview
Bomb Finder is a Minesweeper clone made with C# and raylib.

![](images/screenshot1.png)

![](images/screenshot2.png)

![](images/screenshot3.png)

## Commands to Run
* git clone https://github.com/CodeOnARaft/BombFinder.git
* cd bombfinder
* dotnet run
